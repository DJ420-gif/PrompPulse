What Anthropic's Safety Warning Revealed

Anthropic published an unusually candid warning on red.anthropic.com about Mythos Preview's offensive cybersecurity capabilities. The headline finding: Mythos Preview fully autonomously identified and exploited CVE-2026-4747 — a 17-year-old remote code execution vulnerability in FreeBSD's NFS implementation that allows any unauthenticated internet user to gain root access. No human was involved after the initial request to find the bug. The model found the vulnerability, developed the full exploit chain and achieved privilege escalation without human guidance. For comparison: Claude Opus 4.6 had a near-0% success rate at autonomous exploit development on the same class of tasks.

02

The Firefox Benchmark — 181 vs 2

Anthropic ran a controlled benchmark: take known vulnerabilities in Firefox 147's JavaScript engine (all patched in Firefox 148) and develop working JavaScript shell exploits. Opus 4.6 succeeded twice out of several hundred attempts. Mythos Preview succeeded 181 times and achieved register control on 29 additional attempts. This is not incremental improvement — it is a categorical capability shift. Anthropic explicitly states Mythos Preview is in a different league. The company's internal OSS-Fuzz benchmarks are now largely saturated by Mythos Preview — the benchmarks designed to measure it can no longer distinguish its capability ceiling from a hard upper limit.

03

Project Glasswing — Defenders First Release

Rather than a broad release, Anthropic is deploying Mythos Preview through Project Glasswing — a limited programme giving access to critical infrastructure security teams, open source developers and selected vulnerability researchers. The rationale: enable defenders to begin securing the most important systems before models with similar offensive capabilities become broadly available. Anthropic has disclosed CVE-2026-4747 through responsible disclosure. The vulnerability is now patched. The company explicitly acknowledges Mythos Preview creates genuine offensive cybersecurity risk and is structuring the release to ensure defensive applications outpace offensive ones.

04

What This Means for AI Safety

Anthropic's transparency about Mythos Preview's offensive capabilities deserves credit — publishing this level of detail before broad release is exactly the behaviour the AI safety community has argued for. The honest assessment: frontier AI models have crossed a threshold in autonomous offensive cybersecurity that the security community expected but did not have a precise timeline for. The gap between Opus 4.6 (near-0% autonomous exploit success) and Mythos Preview (181 Firefox working exploits, novel FreeBSD zero-day) represents a categorical capability jump. For security teams: the window to harden critical infrastructure against AI-assisted attacks is now, not when such models become publicly available.
